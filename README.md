# BeerGenie Website

Simple one-page website for BeerGenie app, perfect for GitHub Pages hosting.

## Files

- `index.html` - Main website page
- `index.css` - Styling
- `assets/logo.png` - BeerGenie logo

## Setting up on GitHub Pages

1. Create a new GitHub repository (e.g., `beergenie` or `beergenie-website`)
2. Upload all files from this folder to the repository
3. Go to repository **Settings** â†’ **Pages**
4. Select source: **Deploy from a branch**
5. Choose branch: **main** (or **master**)
6. Select folder: **/ (root)**
7. Click **Save**

Your site will be live at: `https://[your-username].github.io/[repository-name]`

## Privacy Policy URL

The privacy policy is included on the page. To link directly to it from App Store Connect, you can use:

`https://[your-username].github.io/[repository-name]/#privacy`

## Customization

- Update App Store badge link when app is published
- Add screenshots/images if desired
- Modify colors in `index.css` (CSS variables at the top)

## Notes

- Logo is already included in `assets/logo.png`
- Privacy policy is embedded in the HTML
- Fully responsive for mobile devices
- Dark theme matches the app design
